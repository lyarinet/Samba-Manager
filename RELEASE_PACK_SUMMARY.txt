🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

Build Release Package:
  $ chmod +x build_release.sh
  $ ./build_release.sh
  → Creates: releases/stable/samba-manager-1.2.0.tar.gz + .zip + checksums

Validate Release:
  $ chmod +x validate_release.sh
  $ ./validate_release.sh
  → Verifies: archives, checksums, files, documentation, git status

Test Installation:
  $ tar -xzf releases/stable/samba-manager-1.2.0.tar.gz
  $ cd samba-manager-1.2.0
  $ python run.py --dev
  → Access: http://localhost:5000

Deploy with Docker:
  $ cd releases/docker
  $ docker-compose up
  → Access: http://localhost:5000

Generate Changelog:
  $ chmod +x generate_changelog.sh
  $ ./generate_changelog.sh
  → Creates: CHANGELOG_DRAFT.md (review and move to releases/stable/)

Publish to GitHub:
  $ chmod +x publish_release.sh
  $ ./publish_release.sh
  → Creates GitHub release, uploads files, publishes notes
